# Trail Weather

Trail Weather is an app that allows users to search for weather conditions by city, state, or zip code. It fetches data from [WeatherAPI](https://www.weatherapi.com/) and displays current conditions along with a 3-day forecast.

## Live Demo

Visit [Trail Weather](https://trail-weather.vercel.app/)

## Features

- **City Search**: Enter a city, state, or zip code to get weather data.
- **Current Conditions**: Displays temperature, humidity, wind speed/direction, UV index, and "feels like" temperature.
- **Forecast**: Shows a 3-day forecast with daily summaries.
- **Dark/Light Modes**: Supports dark and light modes.
- **Error Handling**: Provides user-friendly error messages for invalid inputs.

## Tech Stack

- **Next.js 13+ (App Router)**
- **React**
- **TypeScript**
- **lucide-react**
- **WeatherAPI**
